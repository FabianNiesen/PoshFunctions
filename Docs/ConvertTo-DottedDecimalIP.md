# ConvertTo-DottedDecimalIP

## SYNOPSIS
Returns a dotted decimal IP address.

## SYNTAX

## DESCRIPTION
ConvertTo-DecimalIP takes 32 bit unsigned integer address into a dotted decimal IP address

## EXAMPLES

### EXAMPLE 1
```
ConvertTo-DottedDecimalIP -IPAddress 16885952
```

Would return
192.168.1.1
