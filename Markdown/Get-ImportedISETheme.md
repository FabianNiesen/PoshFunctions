# Get-ImportedISETheme

## SYNOPSIS
Returns imported themes.

## SYNTAX

```
Get-ImportedISETheme
```

## DESCRIPTION
Returns imported themes from the registry

## EXAMPLES

### EXAMPLE 1
```
$Themes = Get-ImportedISETheme
```

Gets the imported ISE themes from the registry and assigns it to Themes
