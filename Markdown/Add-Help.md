# Add-Help

## SYNOPSIS
Adds a template comment based block into the editor in the Powershell ISE.

## SYNTAX

```
Add-Help [<CommonParameters>]
```

## DESCRIPTION
Adds a template comment based block into the editor in the Powershell ISE.

## EXAMPLES

### EXAMPLE 1
```
Add-Help
```
